# SDLPong
My own pong on SDL

# Building
Linux
-----
Dependencies are `SDL2 SDL2_image SDL2_ttf CMake` just install them from the repository.

For Arch Linux `sudo pacman -S sdl2 sdl2_image sdl2_ttf cmake`

Later just
`
mkdir build\n
cd build\n
cmake ..\n
make\n
`
