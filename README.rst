=======
SDLPong
=======

My own pong on SDL

Building
========

Linux
-----
Dependencies are ``SDL2 SDL2_image SDL2_ttf CMake`` just install them from the repository.

For Arch Linux:

  .. code:: bash

     sudo pacman -S sdl2 sdl2_image sdl2_ttf cmake``

For building:

  .. code:: bash

     mkdir build
     cd build
     cmake ..
     make
